Arquitectura

Flujo de datos

1. Admin

Subir contenido

El admin inicia sesión, crea una Revista y sube imágenes de cada página. Se almacenan en storage/app/public/magazines/.

2. Laravel

API / Inertia

Los modelos Magazine y MagazinePage persisten en SQLite. Inertia entrega los datos directamente a React sin build de API separada.

3. Cliente

Visor flipbook

React renderiza el listado de revistas y el componente <Flipbook/> con react-pageflip reproduce el efecto de hojear.
